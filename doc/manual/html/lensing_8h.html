struct lensing

Structure containing everything about lensed spectra that other modules need to know.

Once initialized by lensing_init(), contains a table of all lensed 's for the all modes (scalar/tensor), all types (TT, TE...), and all pairs of initial conditions (adiabatic, isocurvatures...). FOR THE MOMENT, ASSUME ONLY SCALAR & ADIABATIC
